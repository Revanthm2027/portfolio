01 Featured Project

CREDA — AI-Powered Developer Skill Verification Platform Live

AI-powered platform that uses LLMs to interview developers about their own GitHub code — because resumes don't prove anything.

The Problem

Developers list skills on their resume that they've barely used. Traditional screening methods can't verify if someone actually built production code or just followed a tutorial. Recruiters need better AI-powered skill assessment tools.

What I Built

CREDA uses Large Language Models to analyze GitHub repositories, understand code architecture, and generate personalized technical interviews. The AI-driven questions are specific enough that developers can't fake their way through. Built with Next.js, Node.js, PostgreSQL, and the OpenAI API.

What Happened

200+ developers tried it. Most of them came back to update their profiles. The platform handles concurrent AI interviews at scale, proving that LLM-powered assessment tools have real product-market fit.

What I Learned

Writing prompts for one person is easy. Building a prompt engineering pipeline that works reliably across 200 different codebases taught me that production AI systems need robust error handling, prompt chaining, and continuous optimization.
