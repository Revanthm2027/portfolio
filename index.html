01

Prompt-to-Database Insight Generator

Natural language → SQL → structured insight. No analyst required.

The Problem

Business teams spend hours pulling data or waiting for analysts. The bottleneck is always the translation layer — from business question to query to insight.

What I Built

An LLM pipeline that converts natural language into optimized SQL via OpenAI function calling, runs it against a PostgreSQL backend, and returns structured insights with context. Used AI to accelerate the prompt architecture and testing — but designed the system logic and schema myself.

Real Impact

Non-technical users can query live databases in seconds. Removes analyst bottleneck entirely for routine data questions — a genuine force multiplier for small teams.

03

Trust Layer — Browser Extension

Know what a website knows about you — before you interact.

The Problem

Users interact with hundreds of websites without any idea of what data is collected, how they're tracked, or whether the site is trustworthy. Privacy is invisible until it's too late.

What I Built

A browser extension performing real-time website analysis — privacy policy parsing, tracker detection, WHOIS lookups, SSL validation — surfaced as a clean, non-intrusive trust score overlay. Used AI for NLP-based privacy policy parsing; built the detection logic, extension architecture, and scoring model from scratch.

Real Impact

Gives everyday users professional-grade website intelligence before they engage. Protects against dark patterns. Built for real browsing sessions — not just a demo.

JavaScript Browser Extension APIs Web Scraping NLP Python
